Host content on IPFS with Stomata

Stomata is a simple and incomplete IPFS gateway that aims to be compatible with some bits of the Pinata API. It also adds support for publishing using IPNS entries. It only supports one hardcoded user and does not support JSON Web Tokens.

Warning! This is probably not a good idea to use in production. Just use Pinata if you do not need to host your own IPFS gateway.
